# Research Documentation

This folder contains research, design documents, and exploratory work for the Simple Eiffel ecosystem.

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `01_library-research/` | Research for potential future `simple_*` libraries |
| `02_ecosystem/` | Ecosystem-wide analysis, library relationships |
| `03_tooling/` | Development tools, infrastructure, publishing |
| `04_vision/` | Big ideas, future directions, industry research |
| `05_design/` | Specific design documents with iterations |
| `06_methodology/` | Development methodology, AI guidance, testing |
| `07_external/` | External integrations, migrations, source analysis |
| `08_contracts/` | Contract theory, MML, specification research |
